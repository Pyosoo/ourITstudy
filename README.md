## IT 스터디 및 프로젝트 구인 게시판
- - -
### 사용 기술
HTML, CSS, Javascript, React, Redux, Google Firebase(Auth, RealtimeDB, Hosting)
- - -
### 주요 기능
- 로그인 (구글, 깃허브)
- 게시판 (지역,직무 별 검색 기능)
- 글쓰기
- 자신의 글 수정 및 삭제
- 스크랩 기능(나의 페이지에서 스크랩 해제 가능 및 모아 보기 가능)
- 댓글 기능(수정, 삭제)
